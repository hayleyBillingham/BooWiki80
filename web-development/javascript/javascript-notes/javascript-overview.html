JavaScript Overview

  • JavaScript is an interperted programming language that conforms to the ECMAScript specification.
  • JavaScript is high-level
  • JavaScript is often just-in-time compiled
  • JavaScript is multi-paradigm
  • JavaScript has curly-bracket syntax
  • JavaScript has prototype based object orientation
  • According to GitHub JavaScript is the top programming language
  • JavaScript is the language of the web supported by all major browsers
  • Tools exist to take JavaScript far beyond webpages
  • You can create native smartphone and tablet apps desktop apps for Windows and Mac and even server-side apps using JavaScript
